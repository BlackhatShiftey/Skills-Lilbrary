# Awesome Design MD Brand Catalog

Use this catalog when the user wants a design inspiration but has not picked a brand yet.

For the latest canonical list, run:

```bash
python scripts/design_md.py list
```

## Selection Heuristics

- Pick `vercel`, `linear.app`, or `stripe` for developer startups and infra tools.
- Pick `notion`, `mintlify`, or `apple` for calm, editorial, product-led pages.
- Pick `warp`, `cursor`, or `supabase` for technical dark-mode interfaces.
- Pick `framer`, `figma`, or `runwayml` when motion and visual drama matter.
- Pick `coinbase`, `wise`, or `stripe` when trust and financial clarity matter.
